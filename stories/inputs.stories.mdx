# Basic Button
Poggle has its version of a basic button, which is simply text and padding. Unlike other libraries our intention is to give the customisation freedom entirely to the user. The properties that these buttons
use are simple and easy to set up.

Let's have a look at a button with no properties:

  <Button>Read More</Button>
</Canvas>

# Hover Effects
Like other components you can easily add your own custom hover effects. First enable `btnHover` then change the `btnHoverBgColor` and/or `btnHoverTxtColor`.

<Canvas>
  <Button btnHover={true} btnHoverBgColor={"red"}>Hover Over Me!</Button>
</Canvas>

# Input Boxes
Input boxes are the quintessential hallmark of any project, no matter what you work on you'll eventually need to use input boxes. By default, the input box will have `autoFocus` set to false and it will have predetermined padding and length.
Like all things in Poggle you can adjust these settings.
